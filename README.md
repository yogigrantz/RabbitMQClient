Before running this source code, please setup the messaging service : RabbitMQ or Service Bus from Azure

And then put the URI string for RabbitMQ and connection string for Service Bus in the appsettings.json

This app has been tested to received queued messages from talenttrack POC website. At this point it just logs the messages to the log file.

But the functionality can be expanded to process records into database, send email, setup more message queues, etc....
